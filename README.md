# 🤖 Jini (Genie) – AI Voice Assistant

Jini (Genie) is a voice-controlled AI assistant built using Python that works completely on voice commands.
The goal of this project is to create a smart assistant that can control the computer and perform tasks in a fast and hands-free way.

## ✨ Features

* 🎤 Voice activation using "Hey Jini"
* 🌐 Open websites like YouTube, Google, LinkedIn, WhatsApp
* 🔍 Perform Google searches instantly
* 🎵 Play songs and videos from YouTube
* 📜 Scroll screen (up/down)
* 🎬 Control videos (play, pause, next, previous)
* 🖱️ Mouse control (cursor movement and click)
* 🗂️ Browser tab control (open, close, switch tabs)
* 📖 Get quick information using Wikipedia
* 🤖 AI responses using Llama 3 (Ollama)

## 🧠 Technologies Used

* Python
* SpeechRecognition
* Pyttsx3 (Text-to-Speech)
* PyAutoGUI (Automation)
* PyWhatKit
* Wikipedia API
* Ollama (Llama 3 AI Model) → https://ollama.com
